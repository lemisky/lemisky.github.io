将自己的域名与vercel绑定

发表于 2020-12-11 更新于 2020-12-12

大学时寝室号 5512, 当时 host 域名比较便宜, 于是购入一个 5512.host, 七十多五年, 真是香呀~

现在还有八个月~

参考:

https://vercel.com/docs/custom-domains

https://vercel.com/knowledge/zero-downtime-migration-for-dns

https://sspai.com/post/63028

在对应域名服务商中添加一条记录即可

可使用如下命令检查:

# dns # vercel # 5512.host # 绑定域名
