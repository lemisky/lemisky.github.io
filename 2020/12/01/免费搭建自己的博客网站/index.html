免费搭建自己的博客网站

发表于 2020-12-01

暂且叫做博客吧, 毕竟大家都这么叫~φ(*￣0￣)

新奇的东西还是多呀, 互联网让人眼花缭乱.

记得以前, 我们想要一个自己的网站, 也许就是简单的写写文章文档, 可能要自己搞个服务器, 然后跑系统(自己做, 或现成的, 或简单的h5特效页面), 要不就只有在别人的博客网上注册账号使用. 大型专门的博客网站, 例如: 博客园, CSDN等一些. 这些的优点就是访问无限制, 缺点就是给你什么样的就是什么样, 没得商量, 没法控制. 如果自己搭服务器的话, 首先要钱, 没有一次 xxx, 终生 yyy, 而且访问速度或流量限制, 还得防着服务器攻击, 根本没法用. 各种麻烦各种不靠谱.

不过怎么说呢, 还是自己知道的少了. 非得流行起来才晓得, 菜(～￣▽￣)～

因为刚刚查了下, github pages 08年就推出了, 那时 win7 都还没出来, 而我接触计算机 还是 win8 出来了之后. 那年年底 12-20 , 微软放出 win10 的消息.

Github Pages

简单科普

  • git

    一个代码版本管理工具, 作者是 Linux之父 - Linus

  • github

    基于git的全球最大开源代码托管社区, 目前老爹是 巨硬 , 哈哈 不懂吧, 其实就是 微软, 为啥叫巨硬呢, 因为它牛逼.

  • GitHub pages

    就是github中的一个功能, 能够分配一个二级域名, 然后空间托管. 简单说就是 你只需要提供代码(静态页面吧), 一毛钱不用, 就可以搭建一个自己的博客网站. 可以定制(就是你的代码了), 不要服务器, 也没有访问不稳定什么的问题.

    当然也是有限制, 不过一般来说, 达不到, 等达到了再说吧（￣︶￣）↗　

很简单

  1. 你肯定用过 git 吧, 你肯定有 github 账号吧~, 对, 有, 继续看

  2. 访问github并登录

    上面这张图主要是获取自己的 github id, 下文统一用我的 lemisky 表示, 自己操作全部换成自己的.

  3. 点击 New, 新建一个

  4. 后面就是上传网站文件了, 如果你已经有网站了, 那么直接将根目录 推送上去就可以了.

    注意: 这只能搭建静态网站博客, 跑不了动态解析的, 比如 php, java, asp.net, python

    只支持3种代码, 哪三种就不需要说了吧

    html, css, js

  5. 之后访问 [你的github id].github.io, 就可以了. eg: lemisky.github.io

  6. 简单几步, 非常简单. 静态博客的话, 有很多…, 去搜下 hexo, 国内用比较多. 也可以参考, 我的小草稿 别人的写的好, 我这个辣鸡

先说下 github pages 的限制吧,

  1. 只有一个, 域名也是 只能自己id
  2. 其他限制应该和 vercel 差不多, 主要是 github 只能搞一个.
  3. 看到这里就知道 vercel 能干嘛了吧? 可以这里预览一下, 也许还可以访问

简单几步, 也很简单

  1. 注册个账号呗. vercel官网, 直接用 GitHub 就行, 授权
  2. 好像个人可以 50 个吧, 不过总共流量有限制. 域名呢, 别人没用, 你就可以用
  3. 工作原理就是, 绑定github仓库(gitlab也可以), 效果和 GitHub pages 一样
  4. 后面就不写, 官网更详细
  5. 总之就是几下就可以了
