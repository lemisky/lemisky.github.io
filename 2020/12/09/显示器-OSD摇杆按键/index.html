显示器 OSD摇杆按键

发表于 2020-12-09

摇杆按键菜单又称为: 五维摇杆

经过自己一番摸索, 总结功能如下:

# 显示器 # 摇杆按键 # 按键 # 摇杆 # OSD # 五维摇杆
